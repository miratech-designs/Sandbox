# Contributing

Thank you for contributing. This repository follows enterprise standards for on-prem console workers.

- Fork the repo and create a topic branch for your change.
- Keep pull requests small and focused.
- Include unit tests for new logic and integration tests where applicable.
- Do not include secrets or credentials in commits.
- Follow the coding standards in `COPILOT_INSTRUCTIONS.md`.
- All PRs must include a description, test plan, and link to the related issue.
- Maintainers will review for security, performance, and observability compliance.
