# OpenCode AI

## Overview

[Home URL](https://opencode.ai)

Command line tool installed via Homebrew

### Getting Started

- Starting a new project, you write /init. It’ll then check all your files and create an AGENTS.md file.

### Two inbuilt agents, you switch between them using Tab (just like Cursor)

- Build mode — For writing code, refactoring, and testing.
- Plan mode — Exploring code for deciding on approaches and planning architecture
